# homebrew-cvault

Homebrew tap for [`cvault`](https://github.com/flatoutsolutions/cvault) — a centralized Claude Code credential vault.

## Install

```bash
brew install flatoutsolutions/cvault/cvault
```

This formula is auto-updated on each `cli-v*` tag pushed to `flatoutsolutions/cvault` via the [`release-cli.yml`](https://github.com/flatoutsolutions/cvault/blob/main/.github/workflows/release-cli.yml) workflow. Do not hand-edit `Formula/cvault.rb` — your changes will be overwritten on the next release.
